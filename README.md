Validation helpers.

## Installation
Install using [npm](http://npmjs.com):
```sh
npm install @ox2/validation --save
```
Install using [yarn](http://yarnpkg.com):
```sh
yarn add @ox2/validation
```

## Usage
```js
import { required } from '@ox2/validation'

```
